# Registers

There may not be any registers with the names 0000 (0) and 1111 (F).

| Register | Description     | Bin  | Hex |
| -------- | --------------- | ---- | --- |
| pc       | program counter | 0001 | 1   |
| sp       | stack pointer   | 0010 | 2   |
| a        |                 | 0011 | 3   |
| b        |                 | 0100 | 4   |
| c        |                 | 0101 | 5   |
| d        |                 | 0110 | 6   |
| e        |                 | 0111 | 7   |
| f        |                 | 1000 | 8   |
| g        |                 | 1001 | 9   |
